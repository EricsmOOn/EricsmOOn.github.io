Markdown

什么是Markdown?

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

为什么我们要学习Markdown?

  • 因为编写的便捷性以及阅读的舒适性，它被广泛用于各种文档的编写。
  • 你可以在Github的任何规范的项目上看到它：一个带有.md后缀的说明文档，这是编写文档的官方语言。
  • 同时本博客的所有文章都基于Markdown编写。
